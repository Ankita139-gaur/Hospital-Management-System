# 🏥 Hospital Management System

A full-stack Hospital Management System built using Django, Python, SQLite, HTML, CSS, and Bootstrap.

## 📌 Overview

This project helps manage hospital operations by providing features for doctor management, patient management, and appointment booking through a user-friendly web interface.

## ✨ Features

### Doctor Management
- Add Doctors
- View Doctor List

### Patient Management
- Add Patients
- View Patient List

### Appointment Management
- Book Appointments
- Manage Appointment Records

### Authentication
- User Login
- Logout
- Protected Dashboard Access

### Dashboard
- Total Doctors Count
- Total Patients Count
- Total Appointments Count

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/hospital-management-system.git
```

2. Move to project folder

```bash
cd hospital-management-system
```

3. Install dependencies

```bash
pip install django
```

4. Apply migrations

```bash
python manage.py migrate
```

5. Run the server

```bash
python manage.py runserver
```

6. Open browser

```
http://127.0.0.1:8000/
```
